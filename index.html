Install

How can I get this to the web?

The app itself is built in NextJS so to deploy it you can use a myriad of ways.
  • Using Netlify
  • Using Vercel
  • Using Heroku
You will need a couple of enviroment variables to get the app up and running: 
  • NEXTAUTH_URL
    This is the URL of your application, if you are using docker it will still be 
    http://localhost:3000
  • NEXTAUTH_SECRET
    needed for session encryption, you can generate a good token in the command line using: 
    openssl rand -base64 32
  • TMDB_KEY
    The app uses the TheMovieDB to get movie info but accounts and API keys are free and unlimited. You can see how to get an API key on their docs 
  • DATABASE_URL
    The place where we will save all your sweet movies.
To get a postgres database there are a couple of options, some free ones are:
  • Using Heroku
  • Using Supabase
After your database is set git clone the github project, set the URL of it in the 
.env
file and run all the migrations to create the structure prisma needs width: 
yarn && npx prisma migrate deploy
